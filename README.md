# 長除法練習

國小三年級 iPad 友善的長除法互動練習網站。

**線上版本** https://huansbox.github.io/aiden-math/

## 功能

- 3 位數 ÷ 1 位數直式除法，逐格填答
- 數字鍵盤 + 實體鍵盤輸入
- 延遲提示：錯 2 次才顯示算式提示，鼓勵先自己想
- 連續答對計數（streak）、星星評分、慶祝動畫
- 響應式：桌機、iPad、iPhone 一屏可見

## 環境設置

```bash
npm install
```

## 開發

```bash
# 執行測試
npm test

# 本地開發伺服器
npm run dev
```
