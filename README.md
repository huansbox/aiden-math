# 長除法練習

國小三年級 iPad 友善的長除法互動練習網站。

**線上版本** https://huansbox.github.io/aiden-math/

## 功能

- 3 位數 ÷ 1 位數直式除法，逐格填答
- 數字鍵盤 + 實體鍵盤輸入
- 延遲提示：錯 2 次才顯示算式提示，鼓勵先自己想
- 連續答對計數（streak）、星星評分、累計星星成就、里程碑徽章、慶祝動畫
- 音效回饋（答對/答錯/完成）
- 每日練習 5 題 + 煙火慶祝 + 自由練習模式
- 魔王挑戰：4→5→6 位數除法，3 條命，通關星星獎勵
- PWA：iOS 加入主畫面後全螢幕使用（無網址列）
- 響應式：桌機、iPad Pro 12.9" 橫式並排、iPhone 一屏可見

## 環境設置

